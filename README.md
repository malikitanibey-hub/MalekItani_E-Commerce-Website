Description:
HomeStyle is a modern e-commerce web application designed for browsing and purchasing home furniture and décor products. The platform provides users with a smooth and interactive shopping experience, allowing them to explore a wide range of items such as chairs, tables, and decorative pieces.

Key Features:

🛍️ Product listing with categories and filters

⭐ Dynamic product ratings display

❤️ Wishlist functionality for saving favorite items

🛒 Shopping cart with real-time updates

🔐 User authentication (login/logout system)

👤 Personalized user experience (welcome message)

📱 Responsive and user-friendly interface

Technologies Used:

HTML5

CSS3

JavaScript (Vanilla JS)

Local Storage (for cart, wishlist, and user data)

Objective:
To build a fully functional front-end e-commerce system that demonstrates core web development concepts, including DOM manipulation, state management, and interactive UI design.
